Vanilla App Template

This is a simple template for a vanilla JavaScript application. It includes links to two example pages: a gallery and a form.

You can use this template as a starting point for your own projects. Just replace the content in the example pages with your own code.

The template includes a basic HTML structure, a link to a CSS file for styling, and a link to a JavaScript file for functionality. You can customize the CSS and JavaScript files to suit your needs.

To get started, click on one of the links below to view the example pages.
